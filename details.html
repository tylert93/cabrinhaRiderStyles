Overview

This application deals with helping users to choose the most appropriate kiting equipment depending on the way they wish to ride. The type of kite, board and even control system you need is changes subject to the style of kitesurfing you want to perform, whether this be going for big air or cruising around in the waves.
The application starts off by displaying all of the kitesurfing products currently produced by Cabrinha, a reputable brand within the industry. The user is able to choose a ‘rider style’ from a dropdown menu and the products which are displayed will vary depending on which style is selected.
A ‘click’ event listener is used to detect whenever a style from the dropdown menu is selected. The ‘.getAttribute()’ and ‘.classList.contains()’ methods are used to determine which of the Cabrinha products are applicable to this style and either hide or show them as required. At the same time the ‘.setAttribute()’ method is used to set an attribute “show” as “yes” or “no” for each product.
For some styles, a particular product category, such as “Foil Kits” will have no products showings at all. The ‘.getAttribute()’ method is used again to determine if any rows have no products showing, and if that entire row will be hidden to remove any un-needed headings.
The code files include 2 JavaScript files, the current one (script.js) and the old one (oldScript.js). This is to show the level of improvement from one to the next. The first time I wrote this application the JS file was ‘WET’ and included some hard coding. The latest version is more elegant and smaller by more than 50%.
